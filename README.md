# On the Robustness of SHAP Explanations in Black-Box Machine Learning Models

## Motivation

Explainability methods like SHAP are increasingly used to interpret machine learning predictions. But can we trust these explanations? If a small change in the model or training data causes SHAP values to shift significantly, the explanations may be unreliable.

## Research Question

> **Do SHAP explanations remain stable when the underlying model or training data undergoes small perturbations?**

Secondary goals:
- Compare explanation stability between interpretable (Logistic Regression) and black-box models (Random Forest, XGBoost)
- Measure variability in feature importance rankings
- Evaluate whether explanations change even when predictive performance remains stable

## Experimental Design

1. **Data pipeline** — Load, clean, and preprocess the UCI Adult Income dataset
2. **Model training** — Train Logistic Regression, Random Forest, and XGBoost
3. **SHAP baseline** — Compute global and local SHAP explanations for each model
4. **Robustness experiments** — Perturb the model/data and measure explanation stability:
   - Model random seed variation
   - Bootstrap resampling of training data
   - Gaussian noise on numeric features
   - Training dataset size reduction (100% → 80% → 60%)
5. **Stability metrics** — Spearman rank correlation, cosine similarity, SHAP variance, Stability Index

## Setup

### Prerequisites

- Python ≥ 3.11
- [uv](https://docs.astral.sh/uv/) (recommended) or pip

### Install dependencies

```bash
# Using uv (recommended) — installs from the lockfile for reproducibility
uv sync --all-extras

# Or using pip
python -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
```

### Install pre-commit hooks

```bash
pre-commit install
```

## Running the Experiment

```bash
python run_experiment.py
```

This single command will:
1. Load and preprocess the dataset
2. Train all models
3. Compute baseline SHAP explanations
4. Run robustness perturbation experiments
5. Compute stability metrics
6. Generate all visualizations
7. Save outputs to `outputs/`

## Dataset

**UCI Adult Income** — Binary classification predicting whether income exceeds $50K/year.
- ~48,842 rows, 14 features (mix of categorical and numerical)
- Source: [UCI ML Repository](https://archive.ics.uci.edu/dataset/2/adult)

### Download

Download `adult.data`, `adult.test`, and `adult.names` from the link above and place them in:

```
data/Raw/
```

The pipeline expects these three files to be present before running.
